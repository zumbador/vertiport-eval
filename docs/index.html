HOW IT WORKS

Two questions. One score.

Most site evaluations ask only whether infrastructure can be built. Vertiport Eval asks both questions simultaneously — and weights them for cargo operations, the eVTOL use case with the clearest near-term economics.

▣
SITE SCORE
Parcel size, FAA airspace class, zoning compliance, FEMA flood zone, and grid capacity — scored against FAA and NREL vertiport standards. Max 75 points.
◎
DEMAND SCORE
Employment density, major destinations, medical facilities, cargo infrastructure, and transit gaps. Three modes: passenger, cargo, or combined.
◈
PRIORITY INDEX
Site × 0.60 + Demand × 0.40. Quadrant placement — PRIME SITE, INFRASTRUCTURE PLAY, DEMAND WITHOUT SITE, or LOW PRIORITY — tells you where to focus.
⬡
PDF REPORT
Full export with satellite map, scoring breakdown, regulatory checklist, investment viability model, and 10-year NPV estimate.
◉
AIRSPACE MAP
Interactive 2D Leaflet map with FAA Class B/C/D overlays. 3D obstacle surface map via Mapbox GL JS with FAA EB 105A approach surfaces.
▦
HELIPORT LAYER
8,211 FAA NASR heliports classified by type. Medical heliports within 500m boost both site and demand scores — a strong proxy for verified infrastructure.
DATA SOURCES

Live government data. No estimates where APIs exist.

Live API calls override LLM estimates for every criterion with a public data source. The LLM scores only what the APIs cannot reach.

CRITERION SOURCE TYPE
Parcel acreage + land class Harris County Appraisal District (HCAD) LIVE
Flood zone FEMA National Flood Hazard Layer (NFHL) LIVE
Elevation USGS 3DEP LIVE
FAA airspace classification FAA NASR — 39 Class B + ~85 Class C + ~65 Class D LIVE
Heliport registry (8,211 national) FAA NASR via OurAirports LIVE
Zoning + land use OpenStreetMap / Overpass API LIVE
Power grid capacity EIA Open Data API v2 LIVE
Distributed energy resources NREL RE Atlas API LIVE
Flying days / climate NOAA 30-year Climate Normals (80+ stations) LIVE
Demand scoring (fallback) Anthropic / OpenAI / Gemini — your key BYOK
BRING YOUR OWN KEY

Your key stays on your machine.

Vertiport Eval uses a large language model to score demand factors that public APIs don't cover. You supply the key — it's stored in your OS user data directory and never transmitted to any server other than the provider you choose.
